Pelajaran Informatika

2. Hosting

Pengertian: Hosting adalah tempat untuk menyimpan file, gambar, video, dan data lain dari sebuah website agar bisa diakses secara online.

Fungsi: Menyediakan ruang penyimpanan dan koneksi server agar website dapat diakses kapan saja.

Jenis Hosting:

  • Shared Hosting
  • VPS (Virtual Private Server)
  • Cloud Hosting
  • Dedicated Server
